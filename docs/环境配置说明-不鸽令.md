## 2. 微信小程序配置

### 2.1 AppID

在微信公众平台注册小程序，获取 AppID。

```
# project.config.json
{
  "appid": "<YOUR_APPID>"
}
```

### 2.2 云开发环境

在微信开发者工具中开通云开发，创建环境。

```
# app.js
App({
  onLaunch() {
    wx.cloud.init({
      env: '<YOUR_CLOUD_ENV_ID>',  // 云开发环境 ID
      traceUser: true
    })
  }
})
```

### 2.3 数据库集合初始化

在云开发控制台创建以下集合：

| 集合名 | 说明 | 索引 |
|--------|------|------|
| activities | 活动表 | `location`（地理位置索引）、`status` + `meetTime`（复合索引） |
| participations | 参与记录表 | `activityId` + `status`（复合索引）、`participantId` |
| credits | 用户信用表 | `_id`（openId，默认主键） |
| transactions | 资金流水表 | `activityId`、`wxPayOrderId` |
| reports | 举报记录表 | `activityId` |

**日历与海报功能所需的额外索引：**

| 集合 | 索引字段 | 用途 |
|------|----------|------|
| activities | `initiatorId` + `meetTime` | getCalendarActivities 发起人月度查询 |
| participations | `participantId` + `createdAt` | getCalendarActivities 参与者月度查询 |
| activities | `status` + `meetTime` | checkConflict 待进行活动查询 |
| transactions | `type` + `status` + `createdAt` | 月度补偿金额查询 |
| credits | `score` | getPosterData 击败百分比计算 |

地理位置索引创建方式：
```
// 云开发控制台 → 数据库 → activities → 索引管理
// 字段：location，索引类型：2dsphere
```

## 3. 微信支付配置

### 3.1 商户号申请

1. 在微信支付商户平台（pay.weixin.qq.com）注册商户号
2. 完成主体认证和银行账户绑定
3. 开通"小程序支付"产品权限

### 3.2 分账配置

1. 在商户平台 → 产品中心 → 开通"分账"功能
2. 添加分账接收方：
   - 平台账户（接收 30%）
   - 发起人需通过 API 动态添加为分账接收方

### 3.3 支付密钥与证书

| 配置项 | 获取方式 | 存放位置 |
|--------|----------|----------|
| 商户号 (mchId) | 商户平台首页 | 云函数环境变量 |
| API 密钥 (apiKey) | 商户平台 → 账户中心 → API安全 | 云函数环境变量 |
| API v3 密钥 | 商户平台 → 账户中心 → API安全 | 云函数环境变量 |
| 商户证书 (apiclient_cert.pem) | 商户平台下载 | 云函数私有目录 |
| 商户私钥 (apiclient_key.pem) | 商户平台下载 | 云函数私有目录 |
| 微信支付平台证书 | 通过 API 获取 | 云函数私有目录 |

### 3.4 云函数环境变量

```json
{
  "WX_MCH_ID": "<商户号>",
  "WX_API_KEY": "<API密钥>",
  "WX_API_V3_KEY": "<APIv3密钥>",
  "WX_NOTIFY_URL": "<支付回调地址>",
  "JWT_SECRET": "<核销码签名密钥，自行生成32位随机字符串>"
}
```

## 4. 腾讯地图配置

### 4.1 申请 Key

1. 登录腾讯位置服务控制台（lbs.qq.com）
2. 创建应用，获取 Key
3. 在 Key 管理中启用"WebServiceAPI"和"微信小程序"

### 4.2 小程序配置

```json
// app.json → permission
{
  "permission": {
    "scope.userLocation": {
      "desc": "用于展示附近活动和到场校验"
    }
  }
}
```

```
// 腾讯地图 SDK 配置
const QQMapWX = require('./libs/qqmap-wx-jssdk.min.js')
const qqmapsdk = new QQMapWX({
  key: '<YOUR_TENCENT_MAP_KEY>'
})
```

## 5. 内容安全配置

微信内容安全接口无需额外申请，使用云开发可直接调用：

```javascript
// 云函数中调用
const result = await cloud.openapi.security.msgSecCheck({
  content: '待检测文本'
})
```

图片安全检测：
```javascript
const result = await cloud.openapi.security.imgSecCheck({
  media: {
    contentType: 'image/png',
    value: imageBuffer
  }
})
```

## 7. 定时触发器配置

在对应云函数目录下创建 `config.json`：

```json
// cloudfunctions/autoArbitrate/config.json
{
  "triggers": [
    {
      "name": "arbitrate-timer",
      "type": "timer",
      "config": "0 */1 * * * * *"
    }
  ]
}
```

```json
// cloudfunctions/executeSplit/config.json
{
  "triggers": [
    {
      "name": "split-timer",
      "type": "timer",
      "config": "0 0 */1 * * * *"
    }
  ]
}
```
